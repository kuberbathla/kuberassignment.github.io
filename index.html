ARCHITECTURE

Data Layer: Real-time ReliefWeb API + sample feeds from UNGM, DevEx, SAM.gov, World Bank, Gates Foundation. In production: RSS/API aggregation via cron job (GitHub Actions or Google Apps Script on a daily timer).

Intelligence Layer: Claude API (Sonnet) scores each opportunity across 5 dimensions geographic fit, sector alignment, service match, funder familiarity, strategic value using IDinsight's organizational profile as the evaluation rubric.

Delivery Layer: Slack webhook for daily digest delivery. PDF export for offline review. Extensible to email (SendGrid/SES) and calendar integration for deadline tracking.

Automation: A GitHub Action or Google Apps Script runs this pipeline daily at 8 AM, fetches fresh opportunities, scores them, and posts the digest to a designated Slack channel zero manual intervention required.
