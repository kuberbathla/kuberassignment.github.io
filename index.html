Slack Integration (Optional)

Paste a Slack Incoming Webhook URL to send the daily digest directly to your team's channel.

How do I get a Slack webhook URL?
  1. Go to api.slack.com/apps and click Create New App From scratch
  2. Name it something like IDinsight Opportunity Finder and pick your workspace
  3. In the left sidebar, click Incoming Webhooks toggle it On
  4. Click Add New Webhook to Workspace at the bottom
  5. Choose the channel you want digests posted to (e.g. #business-development) and click Allow
  6. Copy the webhook URL that appears it looks like https://hooks.slack.com/services/T.../B.../...
  7. Paste it in the field above. That's it!

Each webhook is tied to one channel. To post to a different channel, create another webhook. Anyone in your workspace with admin access can do this in ~2 minutes.

ARCHITECTURE

Data Layer: Real-time ReliefWeb API + sample feeds from UNGM, DevEx, SAM.gov, World Bank, Gates Foundation. Production: RSS/API aggregation via daily cron (GitHub Actions or Google Apps Script).

Intelligence Layer: Claude API (Sonnet) scores each opportunity across 5 dimensions geographic fit, sector alignment, service match, funder familiarity, strategic value using IDinsight's organizational profile.

Delivery Layer: Slack webhook for daily digest. PDF export for offline review. Extensible to email (SendGrid/SES) and calendar integration for deadline tracking.

Automation: A GitHub Action or Google Apps Script runs daily at 8 AM, fetches fresh opportunities, scores them, and posts to Slack zero manual intervention.
